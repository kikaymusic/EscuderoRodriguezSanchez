## Descripción
- **Aprendizaje en el bandido de k-brazos:** El problema del Bandido de k-brazos es el mejor ejemplo del dilema de Exploración-Explotación en el contexto del aprendizaje por refuerzo. Suponiendo que tenemos un bandido que puede tomar k acciones, el objetivo es encontrar un algoritmo que consiga un equilibrio entre explorar todas las diferentes opciones y explotar la acción que mejores resultados nos ha generado hasta el momento. Para esta parte crearemos varios bandidos utilizando diferentes familias de algoritmos y distribuciones.
